# 技能：逆向拆解一家公司靠不靠谱

- 场景：尽调
- 来源：示例
- 置信度：★★★★★

## 方法思路
1. 先摸清全貌——财务 + 产品 + 团队 + 竞争，多源交叉，不信单一口径
2. 找核心增长回路——什么在驱动增长？抽出来，再问"这条断了会怎样"
3. 拆单位经济——一笔生意到底赚不赚钱，把隐性成本还原回去
4. 解剖叙事——公司对外讲的故事，和地面事实差多远
5. 综合判定——给一个明确结论 + 置信度，不给模棱两可的中间答案
6. 自检——逐条问"这个结论在什么情况下会被推翻"

## 踩过的坑
- 只盯财务、忽略组织/文化——增长的裂缝常先出现在人上
- 分不清 GAAP 和"调整后利润"——有些公司用调整后数据做 PR
- 把营销术语当护城河——必须追问"这是真壁垒还是品牌包装"

## 可复用模式
- **护城河真伪检测**：客户有没有把业务逻辑映射进平台？映射进去 = 真护城河；只是个仪表盘 = 假
- **增长质量三问**：(1) 增长来自新客户还是老客户扩展？(2) 留存率升还是降？(3) 获客成本增还是减？
- **叙事四层解剖**：概念标签 → 案例故事 → 情感信任 → 立场认同——一层层剥，看哪层开始虚
