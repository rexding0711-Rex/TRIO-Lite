# 🗂 搭建你自己的知识库

> TRIO 有个小秘密：它越用越懂你。但前提是你得给它一个"记东西的地方"。

## 是什么

TRIO 的「知识库」就是一个文件夹——里面放着你看过的公司、分析过的行业、积累的经验。下次遇到类似的事，TRIO 会先翻翻这里：「诶，这个我们之前见过！」

## 怎么建

### 第一步：创建一个数据目录

随便放哪里都行。比如：
- Windows: `D:\我的TRIO数据\`
- Mac/Linux: `~/TRIO数据/`

### 第二步：告诉 TRIO

运行 `bash init.sh`，它会问你「东西放哪？」——把你的目录路径告诉它就行。

或者直接编辑 `config/paths.conf`：
```
TRIO_DATA_DIR=/你的/数据/目录
```

### 第三步：目录长这样

TRIO 会自动帮你建好这些文件夹：

```
你的数据目录/
├── 知识库/
│   ├── company-benchmark/   ← 分析过的公司放这里
│   ├── person-benchmark/    ← 关注的人物放这里
│   ├── industry/            ← 行业研究报告
│   └── methods/             ← 自己总结的方法论
├── 报告/                    ← 生成的报告存这里
├── 归档/                    ← 不常用的挪到这里
└── INDEX.md                 ← 自动维护的目录索引
```

### 第四步：开始积累

每次用 `/尽调` 或 `/竞品`，TRIO 都会自动把分析结果存进去。不用手动整理。

用着用着，你就会发现：
- 第三次分析同行业的公司——它已经知道这个行业的套路了
- 第五次跑 /速判——它开始能直接给结论，不用从头搜
- 攒了 20 家公司——它可以做跨公司对比了

## 这就是「越用越聪明」的秘密

不是什么黑科技——就是记性好、翻得快、不会忘。

> 💡 数据 100% 在你电脑上。TRIO 不会上传任何东西。
