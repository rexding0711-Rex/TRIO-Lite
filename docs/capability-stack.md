# TRIO 日常办公能力

> 聚焦日常办公流。删掉的不是不重要——是不该现在占你的认知空间。

## 📝 写作与交付

| 能力 | 说明 |
|------|------|
| 文档撰写 | Markdown/Word/PPTX 生成 |
| 报告编排 | 结构化长篇报告 |
| 内容校验 | 事实核查 + 逻辑审查 |
| 多格式导出 | MD → PDF/HTML |

## 🔍 调研与分析

| 能力 | 说明 |
|------|------|
| 快速评估 | 10-15分钟速判 |
| 深度尽调 | 含逆向工程、因果分析 |
| 竞品分析 | 赛道扫描 + 对标 |
| 信息搜集 | 多源交叉验证 |

## 🛡️ 质控

| 能力 | 说明 |
|------|------|
| 事实核查 | 来源追溯、置信度标注 |
| 逻辑校验 | 矛盾检测、反事实检验 |
| 交付审计 | 格式完整性、承诺兑现检查 |

## 🧠 记忆与进化

| 能力 | 说明 |
|------|------|
| 知识留存 | 公司库、方法论积累 |
| 行为画像 | 使用习惯、思维偏好 |
| 每日刷新 | 知识消化 + 架构微调 |

## 🎬 表达

| 能力 | 说明 |
|------|------|
| 三面具对话 | 实干家/较真鬼/好奇宝宝 |
| 提要与总结 | 长文压缩为核心要点 |
| 论证呈现 | 结论 + 证据 + 置信度 |
