# /竞品 <目标>

想看看某个产品的竞争对手是谁？三个小伙伴帮你摸一圈。

## 怎么做

### 出发前
- 看看我们有没有分析过类似赛道（翻翻知识库~）
- 有之前的 run 没跑完的话，问你要不要继续

### 三个小伙伴接力
1. **好奇宝宝** — 上网搜一圈：谁在做类似的事？市场份额怎么样？
2. **较真鬼** — 逐项对比：技术、定价、团队、融资——还要求至少找出一个竞品之间的数据矛盾
3. **实干家** — 综合判断：威胁大不大？策略建议是什么？

### 最后
- 较真鬼再做一轮终审，有没过的就修（最多三轮）
- 竞品数据自动存进知识库，下次遇到同赛道直接对比
- 打包一份竞品矩阵给你

> 💡 用 `/尽调 A vs B` 可以把两家放一起深度对比
