# /日进化

每天花几分钟，让 TRIO 陪你回顾今天做了什么、学到了什么、有什么可以变得更好的。

## 怎么做

### 第一步：今天学了什么
- 翻翻今天聊过的话题，看看有没有值得记住的新发现
- 有的话，记到知识库里，下次就能直接用啦

### 第二步：看看最近的自己（每周一深度聊聊）
- 翻翻最近的聊天记录——有没有什么新习惯？新偏好？
- 发现新的能力？悄悄记下来
- 连续好几天没聊天？提醒你一下~

### 第三步：检查一下状态
- 运行 `bash mgmt.sh guard` 看看有没有放错地方的文件
- 运行 `bash mgmt.sh depth` 调整一下输出深度（太长还是太短？）
- 看看各个项目——有超过一周没动的吗？提醒你
- 看看即将到期的里程碑——未来 3 天的标 🔴，7 天的标 🟡

### 第四步：每日学习 + 小建议
- 更新连续使用天数 🔥
- 记录"今日发现"
- 如果有架构改进的小建议，列出来给你看（不会自己改，等你点头）

## 输出格式
```
📅 TRIO 日进化 — YYYY-MM-DD HH:MM

📚 今天学了什么
  知识库: 总共 X 个主题 | 需要关注: N 个
  今天新发现: [一句话]

👀 最近的自己
  今天聊了: N 轮
  新习惯/新模式: [如有]
  能力变化: [如有]

🔍 检查一下
  ✅ [项目] — X 天前聊过 — 一切正常
  🔴 [项目] — X 天前聊过 — 好久没动了，要不要看看？

💡 小建议
  连续使用: 🔥 X 天
  今日发现: [一条洞察]
  明天可以试试: [一条行动建议]
```

## 什么时候跑
- 每天自动跑第一步（如果你设了 cron 的话）
- 手动 `/日进化` 跑全部四步
- 深度聊天（第二步）每周一自动做一次
