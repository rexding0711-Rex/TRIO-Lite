# /trio-run <任意描述>

不想走固定流程？那就随便聊聊——告诉我想做什么，我来决定怎么帮你。

## 我会先判断一下

- 「这像是尽调场景耶，要不要走 /尽调？」—— 命中已有流程，建议你走标准命令
- 「有点像尽调但不太一样」—— 加载最接近的流程，灵活调整
- 「完全新的需求！」—— 想想要找哪几个小伙伴帮忙
- 「嗯……你到底想干嘛？」—— 先问你 2-3 个问题搞清楚

## 复杂的事自动拆开

如果事情比较复杂，我会帮你拆成几个小步骤，画个图给你看。你点头了再开始做。

拆的原则：
- 每个小步骤不超过 5 分钟
- 一个步骤挂了不拖累其他的
- 能同时做的就同时做

## 怎么搭配小伙伴

- 随便问问 → 找一个就够了
- 需要搜资料 + 分析 → 好奇宝宝先手 → 实干家收手
- 需要批判验证 → 较真鬼先手 → 实干家收手
- 需要创作打磨 → 好奇宝宝 → 较真鬼核查 → 实干家排版

> 💡 默认只出方案给你看（不动手）。想直接开始？加 `--execute`
